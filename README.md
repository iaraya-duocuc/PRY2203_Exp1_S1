# SpeedFast - Pedidos en Java

Proyecto Java que simula un sistema de pedidos para una empresa de delivery.

## Tipos de pedidos

* Express
* Comida
* Encomienda

Cada tipo tiene reglas propias para asignar repartidor y calcular el tiempo de entrega.

## Qué se usa

* Herencia
* Interfaces
* Clases abstractas
* Polimorfismo
* Listas (`List`)

## Ejecución

La clase `Main` crea pedidos, los procesa y muestra el historial por consola.

## Estructura

```
config   -> configuraciones y enums
model    -> logica de negocio
ui       -> ejecucion del programa
```
